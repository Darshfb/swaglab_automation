# SauceDemo Quality Engineering Platform 🚀

The ultimate, full-scale Quality Engineering Platform capable of validating frontend DOM behaviors, backend REST APIs, and core service contracts within a single ecosystem.

## 🛠️ Enterprise Technology Stack
* **Language:** Java 11+
* **Frontend Automation:** Selenium WebDriver 4.21.0 (ThreadLocal Parallel Grid)
* **Backend Automation:** RestAssured 5.4.0
* **Contract Testing:** JSON Schema Validator
* **Data Payloads:** Lombok `@Builder` + Jackson Databind
* **Reporting:** Allure (Integrated Screenshots, API Payloads, Environment Metadata)
* **Design Patterns:** Fluent Page Object Model + Component Object + API Builder Pattern

## 📁 Multi-Layer Architecture
```text
src/test/java/com/saucedemo/
├── api/              # RestAssured Base Clients & Builders
├── contract/         # JSON Schema backward-compatibility validation
├── e2e/              # Full Selenium UI end-to-end flows
└── integration/      # Hybrid orchestration (Seed API -> Verify UI -> Teardown API)
```

## 🧠 Platform Capabilities

### 1. Dedicated API Layer
Fully abstracted RestAssured clients with centralized Request/Response specifications. We utilize **Lombok Builders** to dynamically construct massive JSON payloads (e.g., `UserPayload.builder().build()`) instead of relying on static, hard-to-maintain strings.

### 2. Contract Testing
Before any slow UI tests execute, the CI pipeline runs blazing-fast JSON Schema validations against the backend endpoints. If the backend breaks the structural API contract, the pipeline fails instantly, saving massive compute time.

### 3. Hybrid Orchestration (The "Holy Grail")
Why use Selenium to create a user account which takes 10 seconds? The `EndToEndIntegrationTests` utilize APIs for the heavy lifting:
1. Seed the Database instantly via a POST request.
2. Log into the UI via Selenium to verify frontend rendering logic.
3. Automatically execute an API DELETE request to teardown test data, leaving zero trace.

### 4. Automated Allure Observability
Every single HTTP request, HTTP response, query parameter, and JSON payload generated by RestAssured is automatically intercepted by the `AllureRestAssured` filter and visually embedded into the final HTML test reports. Debugging API failures is now instantaneous.

## 🚀 Execution Routing

**Run the Contract & API Layer (Fast Feedback):**
```bash
mvn test -Dgroups="api,contract"
```

**Run the Hybrid Integration Layer:**
```bash
mvn test -Dgroups="integration"
```

**Run the Selenium Grid E2E Layer:**
```bash
docker-compose up -d
mvn test -Dgroups="e2e" -Dbrowser=chrome -DthreadCount=5
```
